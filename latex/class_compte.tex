\doxysubsection{Detailed Description}
Classe abstraite représentant un compte bancaire. 

Un compte possède \+:
\begin{DoxyItemize}
\item un identifiant unique
\item un nom
\item un solde
\end{DoxyItemize}

Il peut contenir \+:
\begin{DoxyItemize}
\item des opérations (revenus / dépenses)
\item des transferts sortants et entrants
\end{DoxyItemize}

Cette classe est destinée à être héritée (ex \+: \doxylink{class_compte_courant}{Compte\+Courant}, \doxylink{class_compte_epargne}{Compte\+Epargne}).
