\doxysubsection{Detailed Description}
Représente une catégorie de dépenses ou de revenus. 

Une catégorie peut être hiérarchique \+:
\begin{DoxyItemize}
\item elle peut avoir une catégorie parente
\item elle peut contenir plusieurs sous-\/catégories
\end{DoxyItemize}

Les catégories sont utilisées pour \+:
\begin{DoxyItemize}
\item classer les opérations financières
\item définir et hériter des budgets 
\end{DoxyItemize}
