\doxysubsection{Detailed Description}
Représente un transfert d\textquotesingle{}argent entre deux comptes. 

Un transfert correspond à un mouvement financier entre \+:
\begin{DoxyItemize}
\item un compte source
\item un compte destination
\end{DoxyItemize}

Les transferts sont utilisés pour \+:
\begin{DoxyItemize}
\item les virements manuels
\item les transferts automatiques en cas de solde négatif
\end{DoxyItemize}

Ils impactent directement le solde des comptes concernés.
