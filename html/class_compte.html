Detailed Description

Classe abstraite représentant un compte bancaire.

Un compte possède :

  • un identifiant unique
  • un nom
  • un solde

Il peut contenir :

  • des opérations (revenus / dépenses)
  • des transferts sortants et entrants

Cette classe est destinée à être héritée (ex : CompteCourant, CompteEpargne).
